# **App Name**: SakaJuntos Web

## Core Features:

- Landing Page: Display a landing page highlighting individual shopping (MiniShopping) and group shopping (Grupos) options.
- MiniShopping Catalog: Enable browsing of MiniShopping products.
- Grupos Promotions: Enable browsing of Grupos (group buying) with a display of ongoing group promotions
- Shopping Cart Display: Display a shopping cart for users to review their selected items before proceeding to a simplified 'checkout'.
- AI Order Summarizer: Use generative AI to summarize order details. The tool decides when or if this feature is active.
- Session persistence: Implement a way to save user sessions.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6), evoking trust and reliability for the e-commerce platform.
- Background color: Very light blue (#F0F8FF), providing a clean and unobtrusive backdrop.
- Accent color: Soft green (#90EE90), used sparingly to highlight calls to action, subtly encouraging user engagement without overwhelming the user.
- Body and headline font: 'PT Sans', a versatile sans-serif for clear readability in both headings and body text.
- Use simple, clean icons, ensuring they are intuitive and universally understandable to facilitate navigation and interaction.
- A clean, intuitive layout emphasizing key navigation points and product showcases.
- Use subtle transitions for loading new pages/data to create a smoother user experience.
